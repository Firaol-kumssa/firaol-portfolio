## 🚀 Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/Firaol-kumssa/firaol-portfolio.git
cd firaol-portfolio
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
